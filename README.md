# OS_8E88
--------------ขั้นตอนการใช้งานโปรแกรม---------------
1.Run ไฟล์ server_os.java เพื่อเปิดใช้งานฝั่งของ server 
2.Run ไฟล์ ChatClient.java เพื่อเริ่มต้นใช้งานในฝั่ง client (หากไม่เปิดใช้งานตัว server ก่อนจะไม่สามารถใช้โปรแกรมได้ จะ retry ทั้งหมด 5 ครั้งก่อนที่จะ Terminate ไป)
3.เมื่อเปิดใช้งานได้สำเร็๋จ ขั้นแรกจะให้กรอก Name โดยที่ต้องไม่มีเว้นวรรคหรือว่างเปล่า จากนั้นกด Enter
4.โปรแกรมจะแสดงคำสั่งทั้งหมดที่โปรแกรมรองรับ พร้อมทั้ง Format ของแต่ละคำสั่ง มีดังนี้
============================= All Instructions =============================
DM <receiver_name> <message>   # send a direct message to a specific friend
JOIN <#room_name>              # join a chat room
WHO <#room_name>               # see who is in the room
SAY <#room_name> <message>     # send a message to everyone in the room
LEAVE <#room_name>             # leave the room
QUIT                           # exit the program 

5.หากผู้ใช้ป้อนคำสั่งผิดหรือป้อนคำสั่งที่ไม่มี โปรแกรมจะแสดงผลบอก
6.กรณีหากต้องการใช้คำสั่ง SAY ผู้ใช้จำเป็นที่จะต้อง JOIN ห้องที่ต้องการก่อน
7.หากผู้ใช้ออกจากโปรแกรมด้วยคำสั่ง QUIT โปรแกรมจะนำผู้ใช้ออกจากทุกห้องโดยอัตโนมัติ
8.หากผู้ใช้ไม่ได้ออกจากโปรแกรมอย่างถูกค้อง(คือใช้คำสั่ง QUIT) เมื่อครบเวลาที่กำหนดโปรแกรมจะนำผู้ใช้ออกจากระบบอย่างถูกต้องอัตโนมัติ
